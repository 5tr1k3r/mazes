Hotkeys
-------
- `Space` - generate a maze
- `Enter` - find a path
- `Esc` - quit

Custom Config
--------------
You can create `custom_config.toml` file in the root folder and override some settings without having to edit `config.py`.
For example:
```
maze_width = 10
maze_height = 15

cell_size = 20
wall_width = 10
```
